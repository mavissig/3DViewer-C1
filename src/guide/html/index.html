Программа для просмотра 3D моделей в каркасном виде (3D Viewer)

Программа разработана на языке Си стандарта C11 с использованием компилятора gcc. В проекте использованы дополнительные библиотеки и модули QT

Программа разработана в соответствии с принципами структурного программирования При написании кода придерживались Google Style

Основные части проекта:

  • Парсинг файла
  • Расчет смещения
  • Отрисовка

Основные цели для работы с проетом:

  • установка:
    • Для установки проекта и сборки тестов используйте цель all
      make all
    • Для установки проекта без сборки тестов используте цель install
      make install
  • Удаление:
    • Для удаления программы использйте цель uninstall
      make uninstall
  • Запуск программы после установки:
    • Для запуска программы используйте цель run
      make run
  • Тесты:
    • Для сборки тестов без установки используйте цель tests
      make tests
    • Для запуска тестов используйте цель run_test
      make run_test
